# Shenzhen Authoritative News Router - SKILL.md

## 核心定位

本 Skill 是一个**深圳权威新闻源优先检索路由器**。

**不是**：通用搜索引擎、全国新闻聚合器、内容总结工具  
**是**：深圳本地新闻的权威源定向器、原始报道溯源助手、信息质量过滤器

## 详细决策规则

### 规则 1: 触发条件识别

**必须同时满足**：
1. 问题涉及**深圳**（地名、政策、事件）
2. 问题属于**新闻/资讯/报道**类别

**触发关键词库**（见 config.json）：
- 直接触发："深圳新闻"、"深圳本地"、"深圳报道"
- 领域触发："深圳政策"、"深圳民生"、"深圳教育"、"深圳交通"
- 媒体触发："深圳特区报"、"读特"、"深圳新闻网"
- 场景触发："深圳今天"、"深圳突发"、"深圳发生了什么"

**隐性触发**（用户没说"深圳"但明显相关）：
- "南山/福田/罗湖/宝安/龙岗/盐田/龙华/坪山/光明/大鹏 有什么新闻"
- "前海/河套/香蜜湖 最新动态"
- "深圳湾/大鹏湾 新闻"

### 规则 2: 新闻源选择策略

| 用户问题特征 | 首选源 | 次选源 | 理由 |
|-------------|--------|--------|------|
| 纸媒原文/党报/政策发布 | 深圳特区报电子版 | 深圳新闻网 | 机关报权威性最高 |
| 即时新闻/今日动态 | 读特 | 深圳新闻网 | 客户端更新最快 |
| 综合资讯/城区动态 | 深圳新闻网 | 读特 | 门户覆盖最全 |
| 突发新闻 | 读特 | 深圳新闻网 | 即时推送能力 |
| 历史报道溯源 | 深圳特区报电子版 | 深圳新闻网 | 纸媒档案完整 |

### 规则 3: 结果排序原则

**第一优先**（最高）：
- 原始报道页面（非转载）
- 官方首发链接
- 纸媒电子版原文

**第二优先**：
- 权威平台的整合报道
- 官方客户端内容
- 政府网站转载

**第三优先**：
- 其他公开网页
- 经核实的转载

**排除**：
- 匿名论坛帖子
- 未经验证的自媒体
- 短视频平台内容（除非官方账号）
- 明显二手转述（无原始出处）

### 规则 4: 回答格式规范

**必须包含**：
1. **来源说明**："根据 [新闻源名称] 报道..."
2. **原文链接**：提供直接链接
3. **内容摘要**：简要概括（不编造）
4. **查看提示**："建议查看原文了解详情"

**可选补充**：
- 为什么选这个源
- 其他相关源的补充信息
- 报道发布时间

**禁止**：
- 不说明来源的笼统表述
- "根据网络信息"等模糊引用
- 编造标题或内容
- 猜测未发布信息

### 规则 5: 降级处理流程

当三个优先源均无结果时：

1. **明确告知用户**：
   > "未在深圳特区报电子版、读特、深圳新闻网找到直接相关报道。"

2. **执行普通搜索**：
   - 扩大搜索范围
   - 标注信息来源

3. **可靠性提示**：
   > "以下信息来自普通网络搜索，建议优先参考上述权威源。"

4. **引导用户**：
   > "如需官方信息，建议直接访问 sznews.com 或 dutenews.com 查询。"

## 安全与边界规则

### 内容安全

1. **不伪造**：绝不编造新闻标题、内容、发布时间
2. **不猜测**：对未发布信息明确说明"暂未找到相关报道"
3. **不误导**：区分"报道"与"评论"、"新闻"与"广告"
4. **不偏见**：客观呈现不同来源信息，不做价值判断

### 来源安全

1. **权威优先**：只使用 config.json 中定义的优先源
2. **匿名排除**：不引用论坛、社群、匿名社交媒体
3. **核实提示**：对争议信息提示用户查看原文
4. **版权尊重**：明确新闻内容版权归原媒体

### 功能边界

1. **不做总结**：本 Skill 是"路由"不是"总结"
2. **不做分析**：不提供深度解读（除非源站有）
3. **不做预测**：不预测未发生事件
4. **不做推荐**：不推荐特定观点或立场

## 与其他 Skill 的协作

| 场景 | 本 Skill 职责 | 其他 Skill 职责 |
|-----|--------------|----------------|
| 深圳新闻 + 需要总结 | 提供权威源链接 | summarize Skill 负责总结 |
| 深圳新闻 + 需要翻译 | 提供中文原文 | translate Skill 负责翻译 |
| 深圳政策 + 需要解读 | 链接政策原文 | 政策分析 Skill 负责解读 |
| 全国新闻 + 深圳部分 | 处理深圳相关内容 | 其他 Skill 处理其他部分 |
